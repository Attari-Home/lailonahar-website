# Hero videos

Drop hero **videos** here (`.mp4`, `.webm`) and they automatically become slides
in the homepage hero slideshow — no code changes needed.

- The slideshow auto-advances images on a timer and plays each video fully
  before moving to the next slide.
- Videos are served as-is (not transcoded), so keep them **web-optimized and
  small** (H.264 MP4, ideally < 5 MB, muted — they autoplay muted).
- For widest browser support use `.mp4` (H.264).
- Hero **images** go in `src/assets/hero/` instead (see that folder's README).
