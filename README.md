# Lail O Nahar Machinery Rentals 🏗️

A modern, ultra-fast static website built for **Lail O Nahar Machinery Rentals**, a heavy equipment and man-crane rental service operating across the UAE (Dubai, Sharjah, Ajman, Abu Dhabi).

This project is engineered strictly for **maximum local SEO performance** and lead generation, utilizing a modern static site architecture. It achieves the smooth, page-less feel of a Single Page Application (SPA) without the heavy JavaScript payload, resulting in perfect Core Web Vitals.

## 🚀 Tech Stack
* **Framework:** [Astro](https://astro.build/) (Static Site Generation)
* **Styling:** Tailwind CSS
* **Routing:** Astro View Transitions (for SPA-like smooth navigation)
* **Assets:** Colocated component assets with automatic WebP conversion
* **Forms:** [Insert Form Provider, e.g., Web3Forms/Netlify Forms] + WhatsApp Floating Widget

## 🎯 Key Features
* **Zero-JS by Default:** Ships raw HTML/CSS to the browser for instant loading.
* **Localized UAE SEO:** Configured with specific metadata and `LocalBusiness` JSON-LD schema for the Gulf market.
* **Next-Gen Asset Optimization:** Heavy machinery imagery is automatically resized and compressed via `astro:assets`.
