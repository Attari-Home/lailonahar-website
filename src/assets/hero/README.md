# Hero images

Drop hero **images** here (`.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`) and they
automatically become slides in the homepage hero slideshow — no code changes
needed. Files are optimized at build time (resized + converted to WebP).

- Slides appear in **filename order**, so prefix with numbers to control order
  (e.g. `01-crane.jpg`, `02-boomlift.jpg`).
- Use **landscape** images (≈1600×900 or larger) for the best full-bleed look.
- Hero **videos** go in `public/hero/` instead (see that folder's README).

Images here are also independent of the gallery (`src/assets/gallery/`).
